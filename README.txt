Estrutura do projeto

- src/: codigo-fonte organizado por responsabilidade.
- reports/: espaco para o relatorio final e figuras.
- results/: saidas geradas pelos experimentos.
- nba_treino.csv e nba_teste.csv: base de dados fornecida.

Como usar este esboco

1. Coloque os arquivos de dados na raiz do projeto, como ja estao neste workspace.
2. Implemente os metodos em src/knn_custom.py e src/kmeans_custom.py.
3. Execute src/main.py para acessar os experimentos.
4. Compare os resultados com scikit-learn usando src/compare_sklearn.py.
